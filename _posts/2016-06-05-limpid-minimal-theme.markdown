Bootstrap based minimal theme for blogs.

Includes:

* Facebook/Twitter/LinkedIn/G+ Share
* Disqus Integration
* Prettify JS Code highlighter
* Font Awesome
* Author block and profile
* Download source facility
* Feature image on posts
