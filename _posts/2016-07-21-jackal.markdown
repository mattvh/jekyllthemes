**[Jackal](https://github.com/clenemt/jackal)** is a very lightweight theme for Jekyll.

## Features
* Intro page
* About page
* Articles, archives and pagination
* Rss feed
* Responsive & great typo
* Code syntax highlighting
* Google Analytics tracking code
